O Spring Boot é um framework criado com o objetivo de simplificar a configuração na construção de aplicações web. Como dito acima, o Spring boot traz diversas vantagens para o desenvolvedor, como uma rápida configuração para o início do projeto, como por exemplo usando o Spring Initializr, agilizando o desenvolvimento de sistemas, além de suportar diversas ferramentas presentes já no Spring, como o cloud, security, entre outros.
A adição de novas dependências ao Spring é feita usando o pom.xml.Na minha opinião hoje a melhor ferramenta para se trabalhar com o Spring é o JetBrains Intellij, que oferece diversas ferramentas nativas e plugins que lidam muito bem o Spring, além de em projetos Maven como um todo. O Spring Initializr é utilizado para a criação dos projetos Spring Boot, criando o projeto maven e adicionando todas as dependências que o desenvolvedor desejar facilmente.
